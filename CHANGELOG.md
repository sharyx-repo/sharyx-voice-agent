# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-18

### Added
- **OSS Foundation**: Root LICENSE (MIT), CONTRIBUTING.md, CODE_OF_CONDUCT.md, and SECURITY.md.
- **GitHub Integration**: CI workflow, Issue templates, and Pull Request template.
- **Testing**: Vitest integration and first unit tests.
- **Documentation**: New professional README.md with quick-start and badges.
- **Modular Architecture**: Reorganized SDK into top-level modules (`/stt`, `/llm`, `/tts`, `/orchestrator`).
- **Environment**: Added `.env.example` for better local setup experience.

### Changed
- **Folder Structure**: Moved `src/providers/*` to top-level `src/` modules for better extensibility.
- **Git Hygiene**: Untracked `node_modules` from the repository and improved `.gitignore`.
